# claude-code-minecraft-clone

A 3D Minecraft clone built with Bun, Three.js, and TypeScript.

## Setup

```bash
bun install
```

## Commands

- `bun run dev` - Start the Vite dev server (http://localhost:5173)
- `bun run build` - Type-check and bundle for production into `dist/`
- `bun run preview` - Preview the production build
- `bun run typecheck` - Run TypeScript type checking

## Controls

- **WASD** - Move
- **Space** - Jump
- **Shift** - Sprint
- **Mouse** - Look around
- **Left click** - Break block
- **Right click** - Place block
- **1-9** - Select hotbar slot
- **Click canvas** - Lock pointer
